# Usage Guide

## JavaScript
```bash
node cups.js
node cups.js 13 14 2
```

## Ruby
```bash
ruby cups.rb
ruby cups.rb 13 14 2
```

## Java
```bash
javac JavaCups.java
# then
java JavaCups
java JavaCups 13 14 2
```
## C
```bash
gcc cups.c -o cups
./cups
./cups 13 14 2
```

## C++
```bash
g++ cups.cpp -o cups
# then
./cups
./cups 13 14 2
```

## PHP
```bash
php cups.php
php cups.php 13 14 2
```
